Why to choose html?

The HyperText Markup Language, or HTML is the standard markup language for documents designed to be displayed
in a web browser. It can be assisted by technologies such as Cascading Style Sheets (CSS) and scripting languages
such as JavaScript.

History of HTML

HTML was created by Sir Tim Berners-Lee in late 1991 but was not -
released officially, published in 1995 as HTML 2.0. HTML 4.01 was
published in late 1999 and was a major version of HTML. HTML is a
very evolving markup language and has evolved with various
versions updating.
